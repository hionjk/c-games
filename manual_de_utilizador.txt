1.Inicie o programa e insira seu nome quando solicitado.
2.Escolha um jogo no menu digitando o número correspondente:
    1 - Cálculos
    2 - Adivinhe o Número
    3 - Pedra, Papel, Tesoura
    4 - Adivinhe a Palavra

    |--------MENU PRINCIPAL--------|
    | Jogo para [Nome do Usuário]  |
    |                              |
    | 1 - Cálculos                 |
    | 2 - Adivinhe o Número        |
    | 3 - Pedra, Papel, Tesoura    |
    | 4 - Adivinhe a Palavra       |
    |                              |
    | S - Sair                     |
    |------------------------------|
Para navegar no menu, escolha a opção desejada digitando o número correspondente ou 'S' para sair.

Após o término de um jogo, você pode escolher jogar novamente, voltar ao menu principal ou sair.
|--------MENU DE FIM DE JOGO--------|
| 1 - Jogar Novamente               |
| 2 - Voltar ao Menu Principal      |
| S - Sair                          |
|-----------------------------------|

Instruções de Uso:
------------------
Cálculos:
    1.O programa mostrará uma expressão matemática, por exemplo, "5 + 3 = ?".
    2.Digite o resultado correto da expressão.
    3.Se sua resposta estiver correta, você ganhará um ponto. Se estiver errada, perderá uma vida.
    4.Objetivo – conseguir 3 respostas corretas antes de perder todas as vidas.

Adivinhe o Número:
    1.O programa escolherá um número entre 1 e 100.
    2.Digite seu palpite.
    3.Receba dicas se o número oculto é maior ou menor que o seu palpite.
    4.Objetivo – adivinhar o número sem perder todas as vidas.

Pedra, Papel, Tesoura:
    1.Escolha uma das opções:
        |--------------------------|
        | 1 - Pedra                |
        | 2 - Papel                |
        | 3 - Tesoura              |
        |--------------------------|
    2.Jogue contra o computador. Regras do jogo: Pedra ganha de Tesoura, 
      Tesoura ganha de Papel, Papel ganha de Pedra.
    3.Objetivo – vencer o computador 3 vezes antes de perder todas as vidas.

Adivinhe a Palavra:
    1.O programa escolherá uma palavra e mostrará como espaços em branco, por exemplo, "_ _ _ _ _".
    2.Digite uma letra para tentar adivinhar a palavra.
    3.Se a letra estiver correta, ela será revelada. Se estiver errada, você perderá uma vida.
    4.Objetivo – adivinhar todas as letras da palavra sem perder todas as vidas.
